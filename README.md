# CSV-Manipulator
>Add your CSV file, edit and save it locally in JSON format
>>This is a work in progress

#### Author
Fabio Di Pane

#### Last Update
31/01/2020

#### Prerequisites
- [Node.js](https://nodejs.org/)

#### Installing
`$npm install`

#### Run server
Inside the src folder: </br>
`$dotnet run`

And go to

> - https://localhost:5001 </br>
or
> - https://localhost:5000
